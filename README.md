# Order Management System (OMS)\n\n## Overview\nThis project is a full-stack Order Management System built with Node.js, Express, React, and PostgreSQL.\n\n## Getting Started\n\n### Prerequisites\n- Node.js (v14 or higher)\n- PostgreSQL (v12 or higher)\n- npm (Node Package Manager)\n\n### Installation\n\n1. Clone the repository:\n   ```bash\n   git clone https://github.com/DavinciShah/bahamut-oms.git\n   cd bahamut-oms\n   ```\n2. Navigate to the backend folder and install dependencies:\n   ```bash\n   cd backend\n   npm install\n   ```\n3. Set up your PostgreSQL database and update `config.js` with your database credentials.\n4. Run database migrations:\n   ```bash\n   npm run migrate\n   ```\n5. Start the backend server:\n   ```bash\n   npm start\n   ```\n\n6. Navigate to the frontend folder and install dependencies:\n   ```bash\n   cd ../frontend\n   npm install\n   ```\n7. Start the React app:\n   ```bash\n   npm start\n   ```\n\n## Folder Structure\n\n- backend/\n  - src/\n    - config/\n    - controllers/\n    - models/\n    - routes/\n    - migrations/\n    - server.js\n  - package.json\n\n- frontend/\n  - public/\n  - src/\n    - components/\n    - services/\n    - App.js\n    - index.js\n  - package.json\n\n## Database Migration\nCreate a migrations folder under `backend/src/` and include SQL files here.
